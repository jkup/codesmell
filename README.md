Codesmell
=========

A stream of complaints about bad code that show up on Twitter.

## Add credentials

Register an app at [apps.twitter.com](https://apps.twitter.com/).

Then put your credentials in index.rb.

## Running the app

    bundle install
    rerun ruby index.rb
